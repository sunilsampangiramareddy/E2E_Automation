*** How to Setup OS Environemnt variables: ***
1) Open cmd prompt and trpe below commands
2) set PASSWORD=my_secure_password 
3) echo %PASSWORD% 
4) reg delete HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Environment /F /V PASSWORD

***Groups Test scripts execution***
pytest C:\Automation_Project\E2E_Automation\tests_SFDC\TC_Login.py -m master (Test script will execute, as it belongs to master group)
pytest C:\Automation_Project\E2E_Automation\tests_SFDC\TC_Login.py -m smoke (Test script will execute, as it belongs to smoke group)
pytest C:\Automation_Project\E2E_Automation\tests_SFDC\TC_Login.py -m regression (Test script will not execute, as it doesn't belongs to regression group)

***Parallel Test scripts execution***
pytest -n 2 "C:\Automation_Project\E2E_Automation\tests_SFDC\TC_Login.py" "C:\Automation_Project\E2E_Automation\tests_SFDC\TC_CreateOpportunity.py"
